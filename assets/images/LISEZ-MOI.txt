CONSEILS POUR DE BONNES PHOTOS :
============================================================

- Privilégiez des photos professionnelles et lumineuses
- Évitez les photos floues ou de mauvaise qualité
- Les photos de votre propre travail sont idéales
- Format paysage (horizontal) recommandé
- Résolution minimum : 72 dpi pour le web

============================================================
APRÈS AVOIR AJOUTÉ LES IMAGES :
============================================================

1. Assurez-vous que les noms de fichiers correspondent exactement
2. Relancez l'application : flutter run -d chrome
3. Les images apparaîtront automatiquement
